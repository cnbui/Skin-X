How It Works

The importance of early diagnosis in the fight against skin cancer and the necessity. Artificial intelligence (AI) has now developed to the point where it can be used to successfully diagnose cancer earlier than the current techniques. My website (HTML, JavaScript, CSS) uses Tensorflow.js a JavaScript library for training and deploying machine learning models in the web browser and Node.js. I collected 107 of the 10,000 images from the HAM10000 dataset. With those images, I created and trained a model in Microsoft Azure CustomVision.ai.
